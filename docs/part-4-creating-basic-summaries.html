Chapter 5 Part 4: Creating Basic Summaries

Core Question: What does the sample look like in simple terms?

These summaries help us understand who is represented in the data. They are not population estimates — for that, we would need to use the survey design object with weighted functions.

5.1 Summary Table: Sample Size by Country

Instructor Note: This shows all countries. Trainees should adapt the code to summarize by Tier_One_Unit for their country.

sample_summary <- data_weighted %>%
  group_by(CountryName) %>%
  summarise(
    N_children = n(),
    N_households = n_distinct(HHID),
    N_villages = n_distinct(VillageID),
    N_admin_units = n_distinct(Tier_One_Unit),
    Children_per_HH = round(n() / n_distinct(HHID), 1)
  )

5.2 Visualizing Geographic Distribution

ggplot(sample_summary, aes(x = reorder(CountryName, N_children), 
                           y = N_children)) +
  geom_col(fill = "#0B8A8F") +
  coord_flip() +
  labs(title = "Child Sample Size by Country",
       subtitle = "Number of children aged 5-16 years assessed",
       x = "Country",
       y = "Number of Children") +
  theme_minimal()

5.3 Creating a Variable Inventory

Let’s create a useful reference by grouping variables into categories:

# First, let's just look at all variable names grouped logically
# We'll identify patterns by looking at prefixes

# Design/Admin variables
design_vars <- c("CountryName", "Tier_One_Unit", "VillageID", "Location", 
                 "HHID", "ChildID", "HH_Weight_Provided", "SubmissionDate", "duration")

# Count variables by prefix pattern
category_counts <- tibble(
  category = c("Design/Admin", "Household (hh)", "Child (ch)", "Parent (pt)",
               "Literacy (l)", "Numeracy (n)", "Outcomes (MPL)"),
  count = c(
    length(design_vars),
    sum(str_starts(names(data_weighted), "hh")),
    sum(str_starts(names(data_weighted), "ch")),
    sum(str_starts(names(data_weighted), "pt")),
    sum(str_starts(names(data_weighted), "l[0-9]")),
    sum(str_starts(names(data_weighted), "n[0-9]")),
    sum(str_starts(names(data_weighted), "MPL"))
  )
)
