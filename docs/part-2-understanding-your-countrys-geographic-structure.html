3.4 Understanding Your Sample’s Hierarchy

Let’s explore how the sample is structured in YOUR country:

# Summary of your country's sample structure
data_weighted %>%
  group_by(CountryName) %>% # no need for this if you have only one country data
  summarise(
    country = unique(CountryName),
    n_regions = n_distinct(Tier_One_Unit),
    n_villages = n_distinct(VillageID),
    n_households = n_distinct(HHID),
    n_children = n(),
    children_per_hh = round(n() / n_distinct(HHID), 1),
    children_per_village = round(n() / n_distinct(VillageID), 1)
  )
